## Execução:
### pré-requisitos:
- Noje.js
- npm
### Instalação:
1. clone o repositório:
  -  git clone https://github.com/MariAssisR/UFMG-eng-soft-2024.git
2. instale dependências:
  - cd cineverse
  - npm install
3. execute localmente:
  - npm start 
<br />
A aplicação será executada no: http://localhost:3000
